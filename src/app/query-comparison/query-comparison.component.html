About

Using this interface, an investigator can leverage the underlying SHUAFIS algorithm to compare two provided samples for similarity. Similarity is scored using a 40-threshold ident system, wherein a score of 40 represents a 0.01% chance of the provided prints not being a match. A score above 40 can be considered ident, statistically speaking.


Preconditions

  • Supplied files must be in .PNG, .JPEG, .BMP, .TIFF, or .WSQ format.
  • Supplied files should be monocrome.
  • To ensure accuracy, supplied files should be as clear as possible, and if need be, digitally enhanced for clarity.
