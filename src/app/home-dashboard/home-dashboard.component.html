Seton Hill University Automated Fingerprint Identification System
Getting Started Where to start.

Welcome to SHUAFIS

The SHUAFIS client interface is controlled through the Control Panel. Simply click on the menu icon in the top left corner, and it will slide onto the interface. You can immediately begin generating queries by clicking on the appropriate tab in the Control Panel.


Queries And You

Interaction with the underlying SHUAFIS software is achieved through queries. You could think of a query as asking SHUAFIS about the identity of a print sample. There are two types of query that can be performed:

  1. Comparison (1:1) Query

    A comparison query is used to analyze two provided prints for similarity.

  2. Unknown (1:N) Query

    An unknown query is used to to individualize an unknown print by searching the SHUAFIS database for similar profiles.

Similarity is scored using a 40-threshold system, wherein a score of 40 represents a 0.01% chance of the provided prints not being a match. Regardless of the query type, when a print is within the similarity threshold to another, it is said to be "ident". In this context, ident is short for identified. A score above 40 can be considered ident, statistically speaking.


Profiles

Persons indexed by SHUAFIS are stored as profiles. Profiles can be added to the SHUAFIS system by using the Upload Profile interface found in the Control Panel.

 FAQs Frequently asked questions.

What do I need to access SHUAFIS?

SHUAFIS can be accessed by any device running a HTML5 compatable web browser. A virtual network tunnel will also need to be installed to facilitate secure access.


Can I access SHUAFIS using my phone/tablet?

Yes. The SHUAFIS interface has been optimized for seamless mobile use.


What format should print images be in?

Supplied files must be in .PNG, .JPEG, .BMP, .TIFF, or .WSQ format.


Are there guidelines for print files?

Supplied files should be monocrome. To ensure accuracy, supplied files should be as clear as possible, and if need be, digitally enhanced for clarity.
