About

This interface facilitates the additon of new fingerprint profiles of known individuals to the SHUAFIS system.


Preconditions

  • Supplied files must be in .PNG, .JPEG, .BMP, .TIFF, or .WSQ format.
  • Supplied files should be monocrome.
  • To ensure accuracy, supplied files should be as clear as possible, and if need be, digitally enhanced for clarity.
  • For uniformity, dates provided should adhere to the MM/DD/YYYY format.
  • If a demographic field is unknown, it may be left blank for SHUAFIS to assign the field as "default".

Missing Digits

  • If a subject is missing a digit, the supplied file for that digit will be of the index finger from the hand on which the missing digit is found.
  • If an index finger is missing, the supplied file will be that of the thumb of the hand on which the the missing index finger is found.
  • If a whole hand is missing, the supplied files for each digit of the missing hand will be that of the index finger of the remaining hand, or the thumb if the index is missing.

Uploading A Profile

  1. Select tencard images saved to your computer using the interface below.
  2. Input demographic infomation for the subject individual, if known.
  3. Click upload. A dialouge should appear with a status confirmation.
